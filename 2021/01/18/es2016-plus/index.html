ES2016+ 新特性索引
2021-01-18   前端, JavaScript

从 ES2015 起，ECMAScript 的标准在每年 6 月份都会正式发布一次作为当年的正式版本。而一种新的语法从提案到变成正式标准，需要经历五个阶段。每个阶段的变动都需要由 TC39 委员会批准。

  • Stage 0 - Strawman（展示阶段）
  • Stage 1 - Proposal（征求意见阶段）
  • Stage 2 - Draft（草案阶段）
  • Stage 3 - Candidate（候选人阶段）
  • Stage 4 - Finished（定案阶段）

一个提案如果能进入 Stage 2，就差不多肯定会包括在以后的正式标准里面。而一个提案如果在当年 6 月之前进入 Stage 4，那么就会在 6 月份正式发布。

这里梳理一下 ES2016 及更高版本的新特性，以供速览。

ES2016

Array.prototype.includes()

判断一个数组是否包含一个指定的值

求幂运算符 **

取代以前的求幂方法 Math.pow()

ES2017

String.prototype.padStart() / String.prototype.padEnd()

给字符串填充字符来达到指定长度

Object.values() / Object.entries()

获取对象的值数组和键值对数组

Object.getOwnPropertyDescriptors()

返回一个对象上自有属性的属性描述符

尾部逗号

在定义或者调用函数时添加尾部逗号

async/await

异步终极解决方案

SharedArrayBuffer / Atomics

增强多线程能力

ES2018

对象的 Rest/Spread 属性

扩展运算符

异步循环

新的 for-await-of 语法

Promise.prototype.finally()

无论结果是 fulfilled 或者是 rejected，在执行 then() 和 catch() 后，都会执行finally 指定的回调函数。

正则表达式优化

  • 正则表达式反向断言
  • 正则表达式命名捕获组
  • 正则表达式新转义符 \p \P
  • 正则表达式 /s 使元字符 . 能匹配包括换行符在内所有字符模式

ES2019

Array.prototype.flat() / Array.prototype.flatMap()

平铺数组

可选的 catch 参数

在没有使用 try/catch 的 error 参数时下可以省略括号

Object.fromEntries()

把键值对数组转换为一个对象，与 Object.entries() 相对应

String.prototype.trimStart() / String.prototype.trimEnd()

从字符串的头尾删除空格

Symbol.prototype.description

Symbol 对象的只读属性 description，可选描述的字符串

JSON 超集

JSON 字符串中的换行符如 \u2028 \u2029 不会再导致 JSON.parse失败

更友好的 JSON.stringify()

对于一些超出范围的 Unicode，为其输出转义序列，使其成为有效 Unicode

Function.prototype.toString()

返回函数的精确字符，包括空格和注释

ES2020

可选链操作符

在查询具有多个层级的对象时，不再需要进行冗余的各种前置校验

空位合并操作符

表达式在 ?? 的左侧运算符求值为 undefined 或 null 时，返回其右侧默认值

Promise.allSettled()

接收一组 Promise，并且会返回所有的结果，而不管是 resolved 还是 rejected

String.prototype.matchAll()

为所有匹配的匹配对象返回一个迭代器

动态引用

import(module) 函数可以在任何地方调用，它返回一个解析为模块对象的 promise

BigInt

第 7 个原始类型，可安全地进行大数整型计算

globalThis

一种标准化方式访问全局对象，等同于浏览器端的 window，node 的 global

导出模块的命名空间

支持导出模块时命名

明确定义 for-in

  • 遍历不到 Symbol 类型的属性
  • 遍历过程中，目标对象的属性能被删除，忽略掉尚未遍历到却已经被删掉的属性
  • 遍历过程中，如果有新增属性，不保证新的属性能被当次遍历处理到
  • 属性名不会重复出现（一个属性名最多出现一次）
  • 目标对象整条原型链上的属性都能遍历到

import.meta

用来读取 ES Module 特定的元信息，import.meta 本身是个对象，原型为 null
